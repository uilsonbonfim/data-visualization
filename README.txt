This folder is composed by: 
1) 1 README file in .txt
2) 1 Juphyter file in .ipynb
3) 1 Python Script in .py
4) 1 Compressed Folder containg 6 datasets in .csv

The output expeceted from this script is: 
1) New browser window containg the generated chat
2) A html file named index with cam be coppied and embeed on a website

Before start it is necessary:

1) Uncompress the file named 'dataset' or download CSV from https://datacatalog.worldbank.org/search/dataset/0037712/World-Development-Indicators and save in a file named 'dataset'
2) install pandas, matplot and mpld3 libraries. 
Copy the codes and paste in your terminal: 

1) pip install pandas
2) pip install matplotlib
3) pip install mpld3
